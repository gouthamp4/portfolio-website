What I do

Data Engineer

As a Data Engineer, I focus on building and optimizing data pipelines to ensure reliable data flow and accessibility across the organization. My work involves monitoring and troubleshooting data processes, optimizing SQL queries, and managing cloud infrastructure to support high-performance data operations. Collaborating closely with data analysts and scientists, I design and implement data transformations tailored to their project needs.

I also automate workflows using tools like Apache Airflow and ensure data accuracy through rigorous quality checks. By documenting processes and exploring new tools, I help maintain an efficient, scalable data environment aligned with business objectives.

Automation

I automate data workflows and integrate DevOps practices to achieve efficient CI/CD deployment, using tools like Apache Airflow for orchestrating ETL processes and AWS Lambda for serverless automation of data tasks.

I employ GitHub Actions and Jenkins for version control and continuous integration, ensuring that updates to data pipelines are tested and deployed seamlessly. For infrastructure management, I leverage Terraform to automate resource provisioning across AWS environments, while AWS CodePipeline manages CI/CD workflows, enabling consistent, error-free deployments. Together, these automation tools streamline data processing, reduce manual intervention, and support a scalable, reliable data infrastructure.

API Development

I develop REST APIs and microservices to facilitate seamless data access and processing across applications. Using frameworks like FastAPI and Flask with Python, I design APIs that allow real-time data retrieval, updates, and integrations, enabling other applications to interact with data systems efficiently.

I also utilize AWS API Gateway to manage and deploy APIs, ensuring scalability and secure access control. By implementing these APIs and microservices, I create modular, reusable endpoints that streamline data workflows, support data-driven applications, and enhance cross-functional data accessibility across the organization.

My Work

Who I am

I am a skilled Data Engineer with a strong background in designing scalable data architectures, developing robust ETL pipelines, and implementing effective data solutions across AWS and Azure environments. Proficient in tools like Snowflake, Databricks, Apache Spark, Airflow, and DBT, I excel at optimizing data workflows and transforming complex datasets into actionable insights for business intelligence.

In my recent roles, I have designed data Lakehouse architectures using Azure Synapse and Databricks with Medallion architecture, enabling high-performance data processing and real-time analytics. My experience with Snowflake includes setting up automated data ingestion using Snowpipe, designing data models, and using DBT for transformations to enhance data accessibility and query efficiency.

With expertise in creating RESTful APIs using FastAPI and AWS API Gateway and automating deployments through Terraform and CI/CD pipelines with Jenkins and Azure DevOps, I bring a comprehensive understanding of cloud-based data infrastructure. I am passionate about leveraging data to drive strategic decisions, enhance processes, and build scalable, reliable data systems that support organizational success.
